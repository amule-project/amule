## Why the project moved

The `amule-project` organization is owned solely by Gonosztopi, who became unreachable. With no way to update the organization's infrastructure, the maintainers created the new **[amule-org](https://github.com/amule-org)** organization so that aMule could continue.

Active development and all current releases — aMule **3.0.0** and later — now live at **[amule-org/amule](https://github.com/amule-org/amule)**.

## About this repository

This repository is kept online for historical reference only. **New issues, pull requests, and releases are not accepted here** — please use [amule-org/amule](https://github.com/amule-org/amule) instead.
